## What is Isaac?

Isaac is a self-hosted farm and homestead management system designed to run on a Raspberry Pi or any Linux server. It provides a comprehensive dashboard for tracking:

- Plants (trees, crops, garden plants)
- Animals (livestock and pets)
- Equipment and vehicles
- Home and farm maintenance
- Tasks and reminders
- Weather conditions and alerts

The system integrates with weather stations (Ambient Weather), calendars (CalDAV), and provides email notifications for important alerts.

## Architecture

- **Backend**: Python FastAPI with SQLite database
- **Frontend**: React with Vite, TailwindCSS
- **Designed for**: Raspberry Pi, touch screens, mobile-friendly
